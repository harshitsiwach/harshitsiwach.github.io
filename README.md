# 0xUgly // CYPHER TERMINAL

> **Status:** ◉ DEGEN MODE ACTIVE | [Switch to Professional →](professional.html)

A retro TUI-style personal website built with **TuiCss** + **Hermes aesthetics** + **braille animations**.

## 🎨 Vibe

- DOS/BIOS retro panels (TuiCss library)
- Neon cyber colors (green #00FF41, pink #FF10F0, blue #00D4FF, purple)
- Braille rain background animation
- CRT scanline overlay effect
- Glitch text headers
- Cypher decode animations
- Scrolling marquee ticker
- Fixed retro status bar
- Degenerate casual copy (or professional mode if you're a square)

- Responsive retro TUI components
- Real-time project status tracking
- Tech stack showcase in table format
- Contact links (GitHub, X/Twitter, Email)

## 🛠️ Tech

- **TuiCss** — Text-based UI CSS library (Turbo Vision-inspired)
- Vanilla JS for glitch/decode effects
- CSS animations for braille rain, CRT, ticker
- No frameworks, no build step — just open in browser
