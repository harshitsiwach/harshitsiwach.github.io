🤖 RuptureLabs RAG — Enterprise Chatbot Platform PRODUCTION

Architecture highlights: Dockerized microservices, vector similarity search, admin dashboard with analytics, multi-tenant support. Operating at ~$380/month infrastructure cost while serving 600+ active users. 6-week rollout with iterative improvements.

🧬 Peptide/Protein ML Fine-Tuning Pipeline RESEARCH

Qwen3.5-0.8B • Axolotl • LoRA • GGUF

Built an end-to-end pipeline for fine-tuning language models on biochemical data (peptides and steroid compounds). Successfully trained on 651 curated examples using parameter-efficient LoRA adaptation, achieving convergence in ~15 minutes on consumer hardware (MacBook Air M4, 16GB RAM).

Goal: Deploy as a fully offline mobile application using GGUF quantization and llama.cpp, enabling on-device molecular property prediction without internet connectivity. Project demonstrates expertise in domain-specific model adaptation and edge deployment.
